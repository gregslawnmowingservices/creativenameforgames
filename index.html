Stage 3: based on a roblox event from october 21'st 2006.
Stage 4: Based on Tornado Alley Classic, Which is a game Imari had good memories with as a child
Stage 5 is based on Sword Fight On The Hights. a game by a past roblox admin named John Shedletsky
Stage 6 is just a random cloud level.
Stage 7 is based on Roblox HQ, made back in 2008.

Scrapped Concepts

We originally were going to add a sword to the game. although this was scrapped as soon as we all realized that no violence was allowed.

A Health/Damage system was considered. but had the same fate as the sword mechanic.

A Skateboard ability was considered as well, but it was tried time and time again and wouldn't work correctly. which ended in it being scrapped as well.

A Bossfight with John Shedletsky was considered on stage 5, although the "no violence" rule crushed that idea.
